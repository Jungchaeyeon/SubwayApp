# SubwayApp
실시간 지하철 정보 앱

gradle.properties 파일에 서울열린데이터 광장에서 발급받은 인증키를 입력해야합니다.
SEOUL_API_ACCESS_KEY="자신의발급인증키"

<br/>

1. 지하철 역 목록 화면(Main)
2. 지하철 역 상세화면<br/>
